Sobre o Agendamento

O sistema verifica a cada minuto se há notícias agendadas para publicação. Quando a data e hora de publicação de uma notícia agendada é atingida, o sistema automaticamente altera seu status para "Publicada".

Nota: Se o servidor for reiniciado, as notícias agendadas serão publicadas assim que o servidor voltar a funcionar, desde que a data e hora de publicação já tenha passado.
